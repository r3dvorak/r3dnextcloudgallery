# pkg_r3dnextcloudgallery

Joomla 6 package for importing Nextcloud share galleries into Joomla articles and rendering them in the frontend, including optional lightGallery-based lightbox support and YOOtheme source integration.

The package contains two child plugins:

- `plg_fields_r3dnextcloudgallery`
- `plg_system_r3dnextcloudgallery_yootheme`

## Build Flow

1. `D:/1DEV/_tools/04-build-extension.ps1 -All`
2. `D:/1DEV/_tools/05-build-package.ps1`

## Update Channel

Official channel: package feed `pkg_r3dnextcloudgallery.xml`.
